Support for Bockle

Bockle is a word game where players race against a configurable timer to find words in a randomized 4x4 dice grid.

4x4 randomized dice grid 1 to 5 minute rounds Customizable dice and timer alerts

How Bockle Works

  • Start a round and Bockle rolls all 16 dice.
  • Players find as many words as possible before time expires.
  • Words must be built from adjacent dice and each die can be used once per word.
  • After the round, duplicate words score 0 and unique words score by character count.
